All about CSS

we implement css in three ways

  1. Inline CSS
  2. Internel CSS
  3. External CSS

CSS (Cascading Style Sheets) can be applied to HTML documents in three main ways: Inline CSS, Internal CSS, and External CSS. Inline CSS is written directly inside an HTML element using the style attribute. It is useful for applying quick, specific styles to a single element, but it becomes difficult to manage in large projects because styles are mixed with HTML code

Internal CSS is written inside a tag within thedocument. This type is commonly used when a single webpage requires unique styling. It keeps styles separate from the page content while still being contained within the same file. Internal CSS is easier to maintain than inline CSS and allows multiple elements on the page to share the same styles. However, it is not ideal for styling multiple webpages because the styles must be repeated in every file.

CSS SELECTORS

  1. class selector
  2. id selector
  3. Universal selector
  4. tag selector

class selector

A class selector is used to apply the same style to multiple elements. It is represented with a dot (.) before the class name. Class selectors are useful when you want several items to share the same design or formatting. They provide flexibility because one class can be reused many times in a webpage.

id selector

An ID selector is used to style a unique element on a page. It is represented with a hash (#) symbol before the ID name. Unlike classes, an ID should only be used once in a webpage because it identifies a specific section or element. ID selectors are commonly used for special styling and page navigation purposes.

Universal selector

A universal selector is represented with an asterisk (*) and is used to select all elements in a webpage. It is mainly used for applying common styles such as margin resets or font settings across the entire page. A tag selector, also called an element selector, targets elements based on their names. It applies the same style to every occurrence of that particular element type in the document.
